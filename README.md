# jueJin-Search

调用掘金搜索接口，返回主观筛选后的数据。

# Usage

首先保证电脑已存在 node10.0+ 环境，然后

1.拉取本项目

```
git clone https://github.com/ZY2071/JueJin-Search.git
cd JueJin-Search
npm i
node index.js
```

2.或者下载本项目压缩包，解压

```
cd JueJin-Search-master  // 进入项目文件夹
npm i
node index.js
```

# Examples

当启动项目后，可以看到控制台输出

```
Listening on port 3000!
```

之后用浏览器或双击打开根目录下的 jueJinSearch.html 文件

稍微等待即可使用搜索功能
